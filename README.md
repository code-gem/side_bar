### How to install

Clone the repo
```shell
git clone <URL>
```

Cd into the cloned folder and install dependencies
```shell
cd <FOLDER>
npm install
```

Run React, this should open a page in your browser
```shell
npm run start
```

_Note:_ if you don't have ndoejs installed go to [this link](https://nodejs.org/en/download/)
